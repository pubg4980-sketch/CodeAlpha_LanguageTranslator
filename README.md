# Lingo — Language Translation Tool

A web-based language translation tool built for the **CodeAlpha AI Internship — Task 1**.

## Features

- Enter text and select a source and target language (35+ languages, plus auto-detect)
- Translates text in real-time using Google Translate
- Displays the translated text clearly alongside the original
- Copy-to-clipboard button for the translation
- Text-to-speech playback for both source text and translation
- One-click language swap

## How to Use

1. Download `lingo-translator.html`
2. Open it in any browser (Chrome or Firefox recommended)
3. Type or paste text, choose your languages, and see the translation instantly

## Tech Stack

- HTML, CSS, JavaScript (no installation or backend required)
- Google Translate web API for translations
